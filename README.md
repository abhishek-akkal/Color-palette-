# 🎨 Color Palette

A modern Color Palette website built using HTML, CSS, and JavaScript.

This project showcases aesthetic images with matching color palettes, smooth hover effects, glowing animations, and an interactive navbar heart animation.

## 📸 Preview

## ✨ Features

- Extracted color palettes from images
- Modern UI design
- Hover glow effects
- Animated heart interaction
- Fixed responsive navbar
- Smooth animations & transitions

## 🛠️ Tech Used

- HTML
- CSS
- JavaScript
- Font Awesome
